# Student Expense & Budget Tracker

A modern, responsive, and intuitive financial dashboard built for students to manage their personal finances effectively. This application allows users to track their daily expenses, monitor category-wise budgets, set up saving goals, and visualize their spending habits over time.

## Live Demo
👉 **[View Live Application](https://student-expense-budget-tracker-peach.vercel.app)**

## Features
- **Dashboard Overview**: Instantly view your total balance, monthly income, and monthly expenses.
- **Transaction Tracking**: Add, edit, and categorize your income and expenses.
- **Budget Management**: Set up monthly budgets per category (Food, Transport, Shopping, etc.) and get visual indicators of your remaining budget.
- **Savings Goals**: Create goals, add funds over time, and "purchase" them once the target is reached.
- **Data Analytics**: Visualize your spending habits through an interactive line chart and a category breakdown doughnut chart.
- **Data Persistence**: All data is securely stored locally in your browser so you never lose your progress.
- **Customization**: Toggle between Light and Dark mode, and update your profile details and currency preferences.

## Technology Stack
- **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Custom Variables, Flexbox, CSS Grid)
- **Icons**: Bootstrap Icons
- **Charts**: Chart.js for data visualization
- **Deployment**: Vercel

## How to Run Locally
1. Clone this repository
2. Navigate into the project directory
3. Open `index.html` in any modern web browser or run it through a local development server.
4. No build steps or package installations are required!

## Directory Structure
- `/css`: Contains modular CSS files for layout, components, charts, and variables.
- `/js`: Contains modular JavaScript files for handling the UI views and local data management.
- `index.html`: The main entry point of the application.
